# 🐋 Nexus Whale | The Apex Web3 Dashboard

Nexus Whale is a premium, high-performance Web3 wallet dashboard and management interface. Designed for the "Whales" of the crypto world, it combines advanced visual interactions with real-time blockchain data.

## ✨ Key Features
- **Apex Predator 3D Mascot:** An interactive whale mascot that follows your cursor movements using pseudo-3D parallax.
- **Global Whale Alert:** Live-feed notifications for large on-chain transactions (Whale Alerts).
- **Integrated Extension UI:** A sleek, mobile-inspired popup interface for sending ETH, viewing NFTs, and tracking activity.
- **Deep Sea Radar:** A built-in sonar mini-game to "scan" for hidden on-chain treasures with synthesized audio SFX.
- **Real-Time Data:** Live Ethereum Gas Fee tracker and a dynamic Crypto Price Ticker.
- **Global Ready:** Fully localized in English, Indonesian, and Simplified Chinese.

## 🛠️ Tech Stack
- **Frontend:** Vanilla JavaScript, Tailwind CSS, Chart.js.
- **Web3:** Ethers.js (Metamask Integration).
- **Audio:** Web Audio API for synthesized sonar & bubble SFX.
- **Animations:** CSS Keyframes & Custom JavaScript Parallax.

## 🚀 How to Use
1. Clone this repository.
2. Open `index.html` via Live Server.
3. Scroll to the bottom and click **"Awaken The Whale"**.
4. Connect your MetaMask (Ensure you are on a Testnet for safety).
5. Dive into the ocean of Web3!
